Instalacin

Todo debera estar en el requirements.txt

Ejecucin

El puerto por defaulr es el 5000.

Para la primera ejecucin tienen que crear la base de datos poniendo en un shell de python en consola:

$ from tweet_factory import init_db
$ init_db()

Para ejecutar normalmente:

$ python tweet_factory.py

Dada la baja proporcin de tweets geolocalizados y los lmites del API de Twitter no es fcil conseguir un volumen importante de tweets geolocalizados. Un thread se queda recogiendo todos los tweets geolocalizados que sea posible encontrar. Dejen ejecutando el programa para tener tweets en la base de datos local.
